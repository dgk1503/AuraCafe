### Open-Cafe ☕

## Features

*  Clean, modern UI with glassmorphism effects
*  Coffee-inspired color palette (warm browns & oranges)
*  Smooth navigation (Home, Ingredients, About)
*  Reusable components
*  Built with Next.js for performance and scalability

---


##  Tech Stack

* **Framework:** Next.js
* **Styling:**  Tailwind
* **Deployment:** Vercel

---

## Getting Started

Clone the repo:

```bash
git clone https://github.com/dgk1503/OpenCafe.git
cd OpenCafe
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in your browser:

```
http://localhost:3000
```

---

## 📁 Project Structure

```
/app or /pages   → Main routes
/components      → Reusable UI components
/public          → Static assets (images, favicon, etc.)
```
